## 1. 产品概述

### 1.1 产品背景

随着 A 股市场规模不断扩大，个人投资者对股票数据的实时跟踪、趋势分析和决策辅助的需求日益增长。目前市面上的工具存在以下痛点：

- **数据分散**：行情数据、财务数据、新闻资讯分散在不同平台
- **分析门槛高**：普通投资者缺乏专业分析工具，难以从数据中提炼有效信息
- **决策效率低**：缺乏系统化的数据聚合和可视化呈现，决策依赖直觉
- **产业链断层**：大多数工具只提供个股数据，缺乏产业链上下游关联分析，投资者难以从产业视角发现联动机会和爆发增长标的
- **图谱更新滞后**：已有的产业链工具图谱通常为预制静态数据，更新不及时、覆盖产业有限，无法响应新兴产业的快速变化
- **增长识别盲区**：缺少系统化的业绩爆发增长筛选机制，投资者容易错过处于高景气赛道且业绩加速释放的优质公司

StockHelper 旨在打造一款轻量级、易上手的股票数据分析工具，帮助投资者高效获取数据、洞察趋势、辅助决策。

### 1.2 产品定位

**一句话定位**：面向个人投资者的轻量级智能股票数据分析助手。

### 1.3 产品目标

| 目标类别 | 具体目标 |
|---------|---------|
| 业务目标 | 覆盖 A 股全市场股票的数据获取与分析，支持基本的技术指标计算；覆盖至少 10 条核心产业链的图谱构建与增长分析 |
| 用户目标 | 用户能在 5 分钟内完成从数据获取到生成分析报告的全流程；能在 3 分钟内完成产业链-公司-增长的一站式挖掘 |
| 技术目标 | 单次分析任务响应时间 ≤ 10 秒，支持 100+ 只股票批量分析；全市场增长扫描 ≤ 30 秒 |

### 1.4 核心价值

1. **数据一站式获取**：聚合行情、财务、新闻等多维度数据
2. **智能分析**：内置常见技术指标和分析模型，降低分析门槛
3. **可视化报表**：自动生成图文并茂的分析报告，直观展示分析结果
4. **产业链洞察**：用户输入任意产业，系统自动调研构建产业链图谱，打通上下游数据链路，从产业全局视角发现投资机会
5. **增长挖掘**：基于财务维度的多因子模型，精准筛选业绩爆发式增长潜力公司

## 2. 用户画像

### 2.1 目标用户

| 用户类型 | 特征 | 需求 |
|---------|------|------|
| **散户投资者** | 有 1-3 年投资经验，会基本 K 线分析 | 快速查看多只自选股的技术指标，获得买卖信号参考 |
| **投资爱好者** | 对股票感兴趣但缺乏专业知识 | 学习技术分析，理解指标含义，辅助入门 |
| **量化入门者** | 有编程基础，想做量化分析 | 获取结构化数据，验证交易策略，回测想法 |
| **产业投资者** | 有 3 年以上投资经验，擅长基本面分析，关注产业趋势 | 从产业链上下游联动中挖掘业绩拐点和高增长标的，跟踪细分赛道景气度 |

### 2.2 用户场景

#### 场景一：晨间速览
> 小李是一名上班族股民，每天早上开盘前花 10 分钟查看自选股。他希望打开工具就能看到所选股票的最新行情、关键指标变化和今日关注信号，快速判断是否需要调整持仓。

#### 场景二：深度分析
> 老王是退休股民，喜欢研究个股。发现一只感兴趣的目标股票后，他想查看其历史走势、均线系统、成交量变化和财务指标趋势，综合判断是否值得建仓。

#### 场景三：策略验证
> 小张是计算机专业学生，想验证"金叉买入、死叉卖出"这个简单策略在近 3 年的收益表现，需要批量获取历史数据并跑一遍回测。

#### 场景四：产业链掘金
> 老赵是职业投资者，最近关注一个新兴赛道"固态电池"。他直接在工具中输入 `chain research 固态电池`，系统自动搜索全网公开信息，构建出"上游原材料（锆、镧、硫化物）→中游电解质/隔膜→下游固态电池制造→整车应用"的产业链图谱，并自动标注各环节的上市公司。老赵基于调研结果进一步筛选业绩弹性最大的标的。

#### 场景五：新兴产业链探索
> 小林是一名成长股投资者，他在新闻上看到"低空经济"被列为国家战略性新兴产业，但不知道这个产业包含哪些公司。他在工具中输入 `chain research 低空经济`，系统自动完成调研，生成包含"飞行器制造→空管系统→基础设施建设→运营服务"的完整图谱，并列出相关上市公司，帮助小林快速建立对这个新产业的认知框架。

#### 场景六：高增长筛查
> 陈姐有 5 年股龄，每季度财报披露后，她想快速从全市场筛选出"营业收入同比增长 > 30%、净利润同比增长 > 50%、ROE > 15%"的成长股，并结合所在产业链位置判断增长是否具有持续性。

```

├── 4.2 自选股管理 (Portfolio Manager)
│   ├── 添加/删除股票

│   └── 批量操作

│   ├── 趋势指标 (MA、EMA、BOLL)
│   ├── 动量指标 (MACD、RSI、KDJ)
│   ├── 成交量指标 (VOL、OBV)
│   └── 自定义指标组合
├── 4.4 基本面分析模块 (Fundamental Analyzer)
│   ├── 财务指标概览

│   └── 估值水平分析
├── 4.5 可视化报表模块 (Report Generator)
│   ├── K 线图绘制
│   ├── 指标走势图
│   ├── 综合看板
│   └── 报告导出
├── 4.6 实用工具模块 (Utilities)
│   ├── 选股筛选器
│   ├── 市场热力图
│   └── 数据导出
└── 4.7 产业链分析模块 (Industry Chain Analyzer) 🔥
    ├── 产业链按需调研（核心）—— 用户输入产业名，自动搜索构建图谱
    ├── 产业链环节公司映射
    ├── 业绩爆发增长筛选模型
    ├── 关键原材料/中间品价格跟踪
```

### 4.2 功能详情

#### FR-01：行情数据获取

| 项目 | 说明 |
|------|------|
| **描述** | 从数据源获取 A 股历史行情数据和实时行情 |
| **优先级** | P0 |
| **输入** | 股票代码 / 名称、时间范围、周期类型 |
| **输出** | 结构化行情数据（日期、开盘、收盘、最高、最低、成交量、成交额） |
| **数据源** | akshare / tushare / yfinance |
| **约束** | 单次查询周期最多支持 5 年数据；支持日线/周线/月线 |

**成功标准**：
- 正确获取指定股票指定时间段的数据
- 数据字段完整，无缺失值
- 网络异常时自动重试 3 次，失败后给出明确提示

#### FR-03：技术指标计算

| 项目 | 说明 |
|------|------|
| **描述** | 基于历史行情数据计算常见技术指标 |
| **优先级** | P1 |
| **支持的指标** | MA(5/10/20/60)、MACD、RSI(6/12/24)、KDJ、BOLL(20)、OBV |

- 指标可叠加展示在 K 线图上
- 同一指标支持多参数配置

#### FR-06：产业链按需调研与图谱构建（核心差异功能）🔥

> **设计理念**：摒弃预制固定图谱，改为**用户驱动、按需调研**模式。当用户查找一个产业链时，系统优先对该产业链进行自动化调研分析，从公开信息中动态构建产业链图谱，确保数据的时效性和全面性。

| 项目 | 说明 |
|------|------|
| **描述** | 用户输入目标产业后，系统自动进行产业链调研，动态构建从上游原材料到下游终端的完整图谱，并将结果缓存供后续使用 |
| **优先级** | P1 |
| **输入** | 产业名称（如"新能源汽车""光伏""低空经济"），用户可自由输入任意产业，不限于预制列表 |
| **输出** | 产业链调研报告（含图谱结构、各环节说明、代表公司、关键趋势） |
| **调研数据源** | 网络公开信息（行业研报、公司年报、财经媒体、行业协会数据） |

**工作流程**：

```
用户输入 "低空经济产业链"
    │
    ▼
┌─────────────────────────────────────┐
│ ① 产业链调研（自动执行）              │
│   ├── 搜索产业定义与边界              │
│   ├── 识别上下游环节（NLP + 结构化）  │
│   └── 提取各环节代表公司             │
└──────────┬──────────────────────────┘
           ▼
┌─────────────────────────────────────┐
│ ② 图谱构建                          │
│   ├── 生成产业链结构树               │
│   ├── 标注环节关系与依赖              │
│   └── 上市公司按环节归位              │
└──────────┬──────────────────────────┘
           ▼
┌─────────────────────────────────────┐
│ ③ 深度分析                          │
│   ├── 各环节市场规模与增速            │
│   ├── 关键公司财务对比               │
│   ├── 政策与技术驱动因素              │
│   └── 景气度与投资焦点判断            │
└──────────┬──────────────────────────┘
           ▼
┌─────────────────────────────────────┐
│ ④ 产出报告（展示 + 缓存）            │
│   ├── 产业链全景图谱                  │
│   ├── 环节-公司映射列表               │
│   ├── 财务指标对比表                  │
│   └── 投资焦点与风险提示              │
└─────────────────────────────────────┘
```

**成功标准**：
- 用户输入任意产业名称（无需预制），系统能完成调研并输出图谱
- 调研过程全自动，用户无需手动收集信息
- 产业链包含至少 3 个环节（上游/中游/下游），每个环节标注 ≥ 3 家关联公司
- 调研完成后产出结构化报告，含图谱 + 公司列表 + 关键趋势摘要
- 调研结果缓存至本地，用户可随时刷新更新
- 单次调研执行总时间 ≤ 60 秒（含网络请求和分析计算）

**缓存策略**：

| 场景 | 行为 |
|------|------|
| 用户首次查询某产业链 | 执行全量调研，构建图谱 |
| 用户再次查询同一产业链 | 加载缓存，提示上次调研时间，提供「重新调研」选项 |
| 用户手动刷新 | 重新执行调研，更新缓存 |
| 缓存超过 7 天 | 自动提示"该产业链数据已超过 7 天，建议刷新" |

#### FR-07：产业链环节公司筛选与对比

| 项目 | 说明 |
|------|------|
| **描述** | 在产业链调研结果的基础上，展示各环节上市公司的财务数据对比，支持多维排序和筛选 |
| **优先级** | P1 |
| **输入** | 产业链名称 + 环节名称（可选） |
| **输出** | 表格形式展示该产业链（或指定环节）上市公司的核心财务指标对比 |
| **数据源** | 产业链调研结果 + 财报 API 动态获取 |

**成功标准**：
- 自动从调研结果中提取上市公司列表，并获取最新财务数据
- 支持按营收增速、净利润增速、毛利率、ROE 等维度排序
- 标注各公司的产业链地位（龙头/核心/一般）
- 提供「切换报告期」功能（当季/上年同期/环比）

#### FR-08：业绩爆发增长筛选模型

| 项目 | 说明 |
|------|------|
| **描述** | 基于多维财务因子构建筛选模型，从全市场或特定产业链中识别业绩爆发式增长潜力公司 |
| **优先级** | P1 |
| **输入** | 筛选范围（全市场 / 指定产业链）、报告期、增长阈值参数 |
| **输出** | 符合条件公司的排名列表（含各维度评分和综合评分） |

**模型设计**：

| 维度 | 指标 | 权重 | 说明 |
|------|------|------|------|
| **营收增长** | 营业收入同比增长率 | 25% | ≥ 30% 为满分，线性递减 |
| **利润增长** | 归母净利润同比增长率 | 25% | ≥ 50% 为满分，线性递减 |
| **盈利能力** | ROE（净资产收益率） | 15% | ≥ 15% 为满分 |
| **增长质量** | 经营性现金流 / 净利润 | 10% | ≥ 0.8 为满分，排除"纸面利润" |
| **盈利趋势** | 毛利率同比变化 | 10% | 毛利率提升加分，下降减分 |
| **估值匹配** | PEG（市盈率/增速） | 10% | PEG < 1 为满分，估值合理 |
| **季节加速** | 环比增速变化 | 5% | 连续两个季度加速加分 |

**特殊标识**：
- 🔥 **业绩爆发**：营收增速 > 50% 且 净利润增速 > 100%
- 📈 **高增长**：营收增速 > 30% 且 净利润增速 > 50%
- ⚡ **困境反转**：过去两年利润为负，当期转正且增速 > 100%
- 🚀 **增速加速**：连续两季度增速环比提升

**成功标准**：
- 筛选结果包含详细的财务指标数值，而非仅排名
- 每个结果附增长驱动因素标签（如"产品涨价""产能扩张""市占率提升"）
- 支持导出筛选结果 CSV
- 单次筛选全市场执行时间 ≤ 30 秒

## 10. 附录

### 10.1 术语表

| 术语 | 说明 |
|------|------|
| K 线 | 反映股价走势的图表，包含开盘、收盘、最高、最低四个价格 |
| MA (Moving Average) | 移动平均线，用于平滑价格数据识别趋势方向 |
| MACD | 指数平滑异同移动平均线，判断趋势强度和转折点 |
| RSI (Relative Strength Index) | 相对强弱指标，判断超买超卖状态 |
| KDJ | 随机指标，用于判断短期买卖时机 |
| BOLL (Bollinger Bands) | 布林带，衡量价格波动范围和相对高低 |
| OBV (On-Balance Volume) | 能量潮指标，追踪资金流向 |
| 产业链 | 从上游原材料到下游终端的产品/服务流转链条，包含多个关联环节 |
| 业绩爆发 | 营业收入同比增长 > 50% 且 归母净利润同比增长 > 100% 的高增长状态 |
| PEG | 市盈率相对盈利增长比率 = PE / 净利润增速，PEG < 1 通常被认为低估 |
| ROE | 净资产收益率 = 净利润 / 净资产，衡量公司为股东创造价值的能力 |
| 毛利率 | （营业收入 - 营业成本）/ 营业收入，反映产品定价能力和成本控制 |
| 困境反转 | 公司从亏损状态转为盈利，且增速显著超过行业平均的经营改善形态 |
| 景气度 | 特定产业或环节的繁荣程度，综合价格、订单、开工率等指标判断 |
