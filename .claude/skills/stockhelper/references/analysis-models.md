## 技术指标计算公式

| 指标 | 参数 | 交易信号 |
|------|------|---------|
| MA(5,10,20,60) | 收盘价滚动平均 | 股价上穿/下穿均线 |
| MACD(12,26,9) | EMA快-EMA慢→DIF→DEA→BAR | DIF上穿DEA=金叉（买入） |
| RSI(14) | 涨幅均值/(涨幅均值+跌幅均值)*100 | >80超买 <20超卖 |
| KDJ(9,3,3) | RSV→K→D→J | K上穿D=金叉 |
| BOLL(20,2) | 中轨±2σ | 触上轨=超买 触下轨=超卖 |

## 增长筛选因子计算

**营收增长得分** = min(100, (营收增速 / 50%) * 100)，满分 50%+
**利润增长得分** = min(100, (净利增速 / 100%) * 100)，满分 100%+
**ROE 得分** = min(100, (ROE / 15%) * 100)，满分 15%+
**综合评分** = Σ(维度得分 × 权重) / 100

## 产业链瓶颈评分

```
瓶颈评分 = 
  公司数维度(0-4) + 
  技术壁垒维度(0-3) + 
  扩产难度维度(0-2) + 
  客户锁定维度(0-1)
```
